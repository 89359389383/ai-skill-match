# 新機能：ルーティング / コントローラー / サービス（基本設計）

※ 既存 `src/routes/web.php` に追記済み（投稿系はログイン必須の制御を付与）
※ 追加した Controller / Model / Service は以下（実装済み）：
    - Controllers:
        - TopController
        - SkillListingController / SkillOrderController / SkillInquiryController
        - ArticleController / MyArticleController
        - QuestionController / AnswerController
        - ProfileController
    - Models:
        - SkillListing / SkillListingAsset / SkillOrder / SkillReview
        - Article / ArticleLike / ArticleBookmark / Tag
        - Question / Answer
    - Services:
        - SkillListingService / SkillOrderService / SkillInquiryService
        - ArticleService / QuestionService / AnswerService

【重要】認証制御（freelancer / company の両対応）
- 記事投稿・質問投稿は「フリーランスでも企業でも投稿できる」想定のため、
  `auth:freelancer` と `auth:company` の **OR条件** を満たす必要がある。
- Laravel標準の `auth:freelancer,company` では OR にならないため、
  本プロジェクトでは `auth.any:freelancer,company` を追加し、それを利用する。
