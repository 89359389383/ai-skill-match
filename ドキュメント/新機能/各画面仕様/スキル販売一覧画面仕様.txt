# スキル販売一覧画面 機能要件

## 概要
AIスキルを販売するサービスの一覧ページ。ログイン不要で閲覧可能。検索・並び替え・ページネーション機能を備え、ユーザーが目的のスキルサービスを見つけやすくする。

---

## ① ヘッダーセクション

### 目的
- ページの目的を明確に伝える
- スキル出品への導線を提供

### 表示項目
- ページタイトル：「スキル販売」（`text-4xl font-bold`）
- サブタイトル：「プロフェッショナルなAIスキルサービス」
- 「スキルを出品」ボタン（右上）
  - アイコン：Plusアイコン
  - 色：パープル～ピンクグラデーション（`from-purple-600 to-pink-600`）
  - テキスト：白色

### 操作・遷移
- 「スキルを出品」ボタンクリック
  - ログイン済み → `/skills/new` へ遷移
  - 未ログイン → `/login` へ遷移（遷移後に元のページに戻る）

---

## ② 検索・フィルターセクション

### 目的
- ユーザーがスキルを効率的に検索できる

### 表示項目
- 検索フォーム（左側）：
  - プレースホルダー：「スキルを検索...」
  - アイコン：虫眼鏡アイコン（左端）
  - スタイル：白背景・角丸・シャドウ付き
- 並び替えドロップダウン（右側）：
  - オプション：「人気順」「評価順」「価格の安い順」「価格の高い順」
  - デフォルト：「人気順」
  - スタイル：白背景・角丸・シャドウ付き

### 操作
- 検索フォーム入力時：リアルタイムでフィルタリング
  - タイトル・説明文・タグで部分一致検索
- 並び替え変更時：即座にソート実行
  - 人気順：`reviewCount` の降順
  - 評価順：`rating` の降順
  - 価格の安い順：`price` の昇順（¥記号を除いて数値比較）
  - 価格の高い順：`price` の降順

---

## ③ 件数表示セクション（上部）

### 目的
- 検索結果の総件数と現在表示中の範囲を明示

### 表示項目
- テキスト：「{総件数} 件中 {開始位置} - {終了位置} 件表示」
- スタイル：`text-gray-600 text-sm`
- 位置：スキル一覧グリッドの直前

### 表示例
- 「15 件中 1 - 12 件表示」（1ページ目）
- 「15 件中 13 - 15 件表示」（2ページ目）

---

## ④ スキル一覧グリッドセクション

### 目的
- スキルサービスを視覚的に分かりやすく一覧表示

### 表示項目
- グリッドレイアウト：
  - デスクトップ（lg以上）：3カラム
  - タブレット（md）：2カラム
  - モバイル（sm以下）：1カラム
- 1ページあたり：12件表示
- スキルカード（各カード）：
  - サムネイル画像（上部）：
    - アスペクト比：16:9
    - ホバー時：1.1倍に拡大（`scale-110`）
  - コンテンツ部（下部）：
    - 販売者情報：
      - アバター画像（丸型・小）
      - 販売者名（太字・グレー900）
      - 職種（「職種：」ラベル付き・グレー500・小文字）
    - スキルタイトル（太字・2行まで表示・グレー900）
    - スキル説明（2行まで表示・グレー600・小文字）
    - タグ（最大3つ表示）：
      - 背景：パープル50
      - テキスト：パープル600
      - スタイル：角丸・小文字
    - フッター部：
      - 評価表示（左側）：
        - 星アイコン（黄色・塗りつぶし）
        - 評価値（太字）
        - レビュー件数（括弧付き・グレー500）
      - 価格表示（右側）：
        - 価格（太字・大きめ・パープル600）
        - 納期（グレー500・小文字）

### 操作・遷移
- カードクリック → `/skills/{id}` へ遷移
- カードホバー時：
  - 影強調（`shadow-lg` → `shadow-2xl`）
  - 上方向に移動（`-translate-y-2`）
  - タイトル色変更（`text-gray-900` → `text-purple-600`）

### 空状態
- 検索結果0件の場合：
  - 虫眼鏡アイコン（大・グレー300）
  - メッセージ：「スキルが見つかりませんでした」
  - サブメッセージ：「検索条件を変更してもう一度お試しください」

---

## ⑤ ページネーションセクション

### 目的
- 大量のスキルを効率的に閲覧できる

### 表示項目
- 「前へ」ボタン（左端）：
  - アイコン：左矢印（ChevronLeft）
  - 1ページ目は無効化（グレーアウト）
- ページ番号ボタン（中央）：
  - 現在ページ：パープル背景・白文字・シャドウ
  - その他ページ：白背景・グレー文字・ボーダー
- 「次へ」ボタン（右端）：
  - アイコン：右矢印（ChevronRight）
  - 最終ページは無効化（グレーアウト）

### 操作
- ページ番号ボタンクリック → 該当ページへ移動
- 「前へ」「次へ」ボタンクリック → 前後のページへ移動
- ページ移動時：スムーズスクロールでページ上部へ移動

### 表示条件
- 総ページ数が2ページ以上の場合のみ表示
- 総ページ数 = `Math.ceil(総件数 / 12)`

## ⑥ 件数表示セクション（下部）

### 目的
- ページネーション後も現在表示中の範囲を明示

### 表示項目
- テキスト：「{総件数} 件中 {開始位置} - {終了位置} 件表示」
- スタイル：`text-gray-600 text-sm`
- 位置：ページネーションの直下・中央揃え

## レスポンシブデザイン

### デスクトップ（lg以上: 1024px~）
- グリッド：3カラム
- カード幅：自動調整（`grid-cols-3`）
- ヘッダー：横並び（タイトル左・ボタン右）
- 検索フィルター：横並び

### タブレット（md: 768px~1023px）
- グリッド：2カラム
- カード幅：自動調整（`grid-cols-2`）
- ヘッダー：縦並び（ボタン下に配置）
- 検索フィルター：横並び

### モバイル（~767px）
- グリッド：1カラム
- カード幅：100%
- ヘッダー：縦並び
- 検索フィルター：縦並び

## データ取得・ソート・フィルタリング

### データソース
- `mockSkills` から全15件取得

### フィルタリングロジック
```typescript
const filteredSkills = mockSkills.filter((s) => {
  const matchesSearch =
    s.title.toLowerCase().includes(searchTerm.toLowerCase()) ||
    s.description.toLowerCase().includes(searchTerm.toLowerCase());
  return matchesSearch;
});
```

### ソートロジック
```typescript
switch (sortBy) {
  case "評価順":
    return b.rating - a.rating;
  case "価格の安い順":
    return parseInt(a.price.replace(/[^0-9]/g, "")) - 
           parseInt(b.price.replace(/[^0-9]/g, ""));
  case "価格の高い順":
    return parseInt(b.price.replace(/[^0-9]/g, "")) - 
           parseInt(a.price.replace(/[^0-9]/g, ""));
  default: // 人気順
    return b.reviewCount - a.reviewCount;
}
```

### ページネーションロジック
```typescript
const totalPages = Math.ceil(filteredSkills.length / 12);
const startIndex = (currentPage - 1) * 12;
const endIndex = startIndex + 12;
const currentSkills = filteredSkills.slice(startIndex, endIndex);
```

## アクセス制御
- **閲覧権限**：全ユーザー（ログイン不要）
- **出品権限**：ログインユーザーのみ

---

## 共通仕様

### アニメーション
- カードホバー時：`transform: translateY(-8px)` + `shadow-2xl`
- 画像ホバー時：`transform: scale(1.1)`
- トランジション時間：300ms（`duration-300`）

### 色指定
- 主要カラー：パープル（`purple-600`）、ピンク（`pink-600`）
- 背景カラー：グレー50（`gray-50`）、ホワイト（`white`）
- テキストカラー：グレー900（`gray-900`）、グレー600（`gray-600`）
- アクセントカラー：イエロー（評価星）

### フォント
- タイトル：`font-bold text-4xl`
- サブタイトル：`text-gray-600`
- カードタイトル：`font-bold text-lg`
- 本文：`text-sm`
