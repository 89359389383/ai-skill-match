# スキル販売詳細画面 機能要件

## 概要
出品されているスキル・サービスの詳細ページ。サービス内容、価格、納期、レビュー、出品者情報を表示。ログイン不要で閲覧可能、購入・質問はログイン機能想定（現在はフロントエンドのみ）。

## JavaScript機能

### スムーズスクロール
```javascript
function scrollToSection(sectionId) {
  const element = document.getElementById(sectionId);
  if (element) {
    element.scrollIntoView({ behavior: 'smooth', block: 'start' });
  }
}
```

### 購入処理
```javascript
function handlePurchase() {
  alert('購入手続きを開始します');
}
```

### チャット表示切り替え
```javascript
function toggleChat() {
  const modal = document.getElementById('chatModal');
  if (modal.style.display === 'none') {
    modal.style.display = 'block';
  } else {
    modal.style.display = 'none';
  }
}
```

### メッセージ送信
```javascript
function sendMessage() {
  const input = document.getElementById('chatInput');
  const message = input.value.trim();
  if (!message) return;

  const messagesContainer = document.getElementById('chatMessages');
  const now = new Date();
  const timeStr = now.getHours().toString().padStart(2, '0') + ':' + 
                  now.getMinutes().toString().padStart(2, '0');

  // ユーザーメッセージを追加
  messagesContainer.innerHTML += `
    <div class="flex justify-end">
      <div class="max-w-[80%] rounded-xl p-3 bg-orange-500 text-white">
        <p class="text-sm">${message}</p>
        <p class="text-xs mt-1 text-orange-100">${timeStr}</p>
      </div>
    </div>
  `;

  input.value = '';
  messagesContainer.scrollTop = messagesContainer.scrollHeight;

  // 自動返信（1秒後）
  setTimeout(() => {
    const replyTime = new Date();
    const replyTimeStr = replyTime.getHours().toString().padStart(2, '0') + ':' + 
                         replyTime.getMinutes().toString().padStart(2, '0');
    messagesContainer.innerHTML += `
      <div class="flex justify-start">
        <div class="max-w-[80%] rounded-xl p-3 bg-white text-gray-800 border border-gray-200">
          <p class="text-sm">ありがとうございます。お問い合わせを受け取りました。確認後、お答えいたします。</p>
          <p class="text-xs mt-1 text-gray-500">${replyTimeStr}</p>
        </div>
      </div>
    `;
    messagesContainer.scrollTop = messagesContainer.scrollHeight;
  }, 1000);
}
```

### Enterキー送信
```javascript
document.addEventListener('DOMContentLoaded', function() {
  const chatInput = document.getElementById('chatInput');
  if (chatInput) {
    chatInput.addEventListener('keypress', function(e) {
      if (e.key === 'Enter') {
        sendMessage();
      }
    });
  }
});
```
