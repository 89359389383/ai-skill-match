# 記事一覧画面 機能要件

## 概要
コミュニティメンバーが投稿したAI関連記事の一覧ページ。ログイン不要で閲覧可能。検索・並び替え・ページネーション機能を備え、有益な情報に素早くアクセスできる。

---

## ① ヘッダーセクション

### 目的
- ページの目的を明確に伝える
- 記事投稿への導線を提供

### 表示項目
- ページタイトル：「記事」（`text-4xl font-bold`）
- サブタイトル：「AI技術やノウハウに関する実践的な記事」
- 「記事を投稿」ボタン（右上）：
  - アイコン：Plusアイコン
  - 色：グリーン～エメラルドグラデーション（`from-green-600 to-emerald-600`）
  - テキスト：白色
- 「投稿記事一覧」ボタンクリック（ログインユーザーのみ表示）：
  - `/my-articles` へ遷移
  - ボタンスタイル：
    - アイコン：ファイルテキストアイコン（FileText）
    - 色：オレンジ～アンバーグラデーション（`from-orange-600 to-amber-600`）
    - テキスト：白色
    - 位置：「記事を投稿」ボタンの左側

### 操作・遷移
- 「記事を投稿」ボタンクリック：
  - ログイン済み → `/articles/new` へ遷移
  - 未ログイン → `/login` へ遷移（遷移後���元のページに戻る）
- 「投稿記事一覧」ボタンクリック（ログインユーザーのみ表示）：
  - `/my-articles` へ遷移
  - ボタンスタイル：
    - アイコン：ファイルテキストアイコン（FileText）
    - 色：オレンジ～アンバーグラデーション（`from-orange-600 to-amber-600`）
    - テキスト：白色
    - 位置：「記事を投稿」ボタンの左側

---

## ② 検索・フィルターセクション

### 目的
- ユーザーが記事を効率的に検索・ソートできる

### 表示項目
- 検索フォーム（左側）：
  - プレースホルダー：「記事を検索...」
  - アイコン：虫眼鏡アイコン（左端）
  - スタイル：白背景・角丸・シャドウ付き
- 並び替えドロップダウン（右側）：
  - オプション：「最新順」「人気順」「いいね順」
  - デフォルト：「最新順」
  - スタイル：白背景・角丸・シャドウ付き

### 操作
- 検索フォーム入力時：リアルタイムでフィルタリング
  - タイトル（title）で部分一致検索
  - 要約（excerpt）で部分一致検索
- 並び替え変更時：即座にソート実行
  - 最新順：`createdAt` の降順
  - 人気順：`views` の降順
  - いいね順：`likes` の降順

## ③ 件数表示セクション（��部）

### 目的
- 検索結果の総件数と現在表示中の範囲を明示

### 表示項目
- テキスト：「{総件数} 件中 {開始位置} - {終了位置} 件表示」
- スタイル：`text-gray-600 text-sm`
- 位置：記事一覧グリッドの直前

### 表示例
- 「24 件中 1 - 12 件表示」（1ページ目）
- 「24 件中 13 - 24 件表示」（2ページ目）

---

## ④ 記事一覧グリッドセクション

### 目的
- 記事を視覚的に分かりやすく一覧表示

### 表示項目
- グリッドレイアウト：
  - デスクトップ（lg以上）：3カラム
  - タブレット（md）：2カラム
  - モバイル（sm以下）：1カラム
- 1ページあたり：12件表示
- 記事カード（各カード）：
  - アイキャッチ画像（上部）：
    - アスペクト比：16:9
    - ホバー時：1.1倍に拡大（`scale-110`）
    - トランジション：500ms
  - コンテンツ部（下部）：
    - 記事タイトル：
      - フォント：`text-lg font-bold`
      - カラー：グレー900
      - 行制限：2行まで表示（`line-clamp-2`）
      - ホバー時：グリーン600に変更
    - 記事要約：
      - フォント：`text-sm`
      - カラー：グレー600
      - 行制限：2行まで表示（`line-clamp-2`）
    - フッター部：
      - 著者情報（左側）：
        - アバター画像（丸型・小）
        - 著者名（太字・グレー900・小）
      - エンゲージメント情報（右側）：
        - 閲覧数：
          - アイコン：目アイコン
          - 値：`{views}k` 形式（1000以上の場合）
          - カラー：グレー500
          - フォント：`text-sm`
        - いいね数：
          - アイコン：ハートアイコン
          - 値：数値
          - カラー：グレー500
          - フォント：`text-sm`

### 操作・遷移
- カードクリック → `/articles/{id}` へ遷移
- カードホバー時：
  - 影強調（`shadow-lg` → `shadow-2xl`）
  - 上方向に移動（`-translate-y-2`）
  - タイトル色変更（`text-gray-900` → `text-green-600`）

### 空状態
- 検索結果0件の場合：
  - トレンドアップアイコン（大・グレー300）
  - メッセージ：「記事が見つかりませんでした」
  - サブメッセージ：「検索条件を変更してもう一度お試しください」

## ⑤ ページネーションセクション

### 目的
- 大量の記事を効率的に閲覧できる

### 表示項目
- 「前へ」ボタン（左端）：
  - アイコン：左矢印（ChevronLeft）
  - 1ページ目は無効化（グレーアウト）
- ページ番号ボタン（中央）：
  - 現在ページ：グリーン600背景・白文字・シャドウ
  - その他ページ：白背景・グレー文字・ボーダー
- 「次へ」ボタン（右端）：
  - アイコン：右矢印（ChevronRight）
  - 最終ページは無効化（グレーアウト）

### 操作
- ページ番号ボタンクリック → 該当ページへ移動
- 「前へ」「次へ」ボタンクリック → 前後のページへ移動
- ページ移動時：スムーズスクロールでページ上部へ移動

### 表示条件
- 総ページ数が2ページ以上の場合のみ表示
- 総ページ数 = `Math.ceil(総件数 / 12)`

## レスポンシブデザイン

### デスクトップ（lg以上: 1024px~）
- グリッド：3カラム
- カード幅：自動調整（`grid-cols-3`）
- ヘッダー：横並び（タイトル左・ボタン右）
- 検索フィルター：横並び

### タブレット（md: 768px~1023px）
- グリッド：2カラム
- カード幅：自動調整（`grid-cols-2`）
- ヘッダー：縦並び（ボタン下に配置）
- 検索フィルター：横並び

### モバイル（~767px）
- グリッド：1カラム
- カード幅：100%
- ヘッダー：縦並び
- 検索フィルター：縦並び

## データ取得・ソート・フィルタリング

### データソース
- `mockArticles` から全件取得

### フィルタリングロジック
```typescript
const filteredArticles = mockArticles.filter((a) => {
  const matchesSearch =
    a.title.toLowerCase().includes(searchTerm.toLowerCase()) ||
    a.excerpt.toLowerCase().includes(searchTerm.toLowerCase());
  return matchesSearch;
});
```

### ソートロジック
```typescript
filteredArticles = [...filteredArticles].sort((a, b) => {
  switch (sortBy) {
    case "人気順":
      return b.views - a.views;
    case "いいね順":
      return b.likes - a.likes;
    default: // 最新順
      return (
        new Date(b.createdAt).getTime() - new Date(a.createdAt).getTime()
      );
  }
});
```

### ページネーションロジック
```typescript
const totalPages = Math.ceil(filteredArticles.length / 12);
const startIndex = (currentPage - 1) * 12;
const endIndex = startIndex + 12;
const currentArticles = filteredArticles.slice(startIndex, endIndex);
```

## アクセス制御
- **閲覧権限**：全ユーザー（ログイン不要）
- **投稿権限**：ログインユーザーのみ

---

## 共通仕様

### アニメーション
- カードホバー時：`transform: translateY(-8px)` + `shadow-2xl`
- 画像ホバー時：`transform: scale(1.1)`
- トランジション時間：
  - カード：300ms（`duration-300`）
  - 画像：500ms（`duration-500`）

### 色指定
- 主要カラー：グリーン（`green-600`）、エメラルド（`emerald-600`）
- 背景カラー：グレー50（`gray-50`）、ホワイト（`white`）
- テキストカラー：グレー900（`gray-900`）、グレー600（`gray-600`）
- アイコンカラー：グレー500（`gray-500`）

### フォント
- タイトル：`font-bold text-4xl`
- サブタイトル：`text-gray-600`
- カードタイトル：`font-bold text-lg`
- 本文：`text-sm`
