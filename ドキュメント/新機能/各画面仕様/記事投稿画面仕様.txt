# 記事投稿画面 機能要件

## 概要
AI関連の知識や経験を記事として投稿するための入力フォーム画面。ログイン必須。セクション・サブセクション構造を持つ階層的な記事構成を作成でき、プレビュー確認後に公開する。

---

## ① ページヘッダーセクション

### 目的
- 現在の操作内容を明示
- 記事一覧への戻るリンク提供

### 表示項目
- 戻るリンク：「記事一覧に戻る」
  - アイコン：左向き矢印（ArrowLeft）
  - 色：グレー600
  - ホバー：グレー900
- ページタイトル：「記事を投稿」（`text-4xl font-bold`）
- アクションボタン（右上）：
  - 「プレビュー」ボタン：
    - 色：グリーン600ボーダー・グリーン600テキスト
    - スタイル：白背景・2pxボーダー・角丸xl
    - アイコン：目アイコン（Eye）
  - 「投稿する」ボタン：
    - 色：グリーン600～エメラルド600グラデーション
    - スタイル：白文字・シャドウlg・角丸xl
    - アイコン：保存アイコン（Save）
    - ホバー：shadow-xl + translateY(-1px)

## ② 基本情報セクション

### 目的
- 記事の基本情報を入力

### 表示項目
- セクションタイトル：「基本情報」（`text-2xl font-bold`）
- 背景：白背景・角丸2xl・シャドウlg

#### タイトル入力
- ラベル：「タイトル」（必須マーク付き・赤500）
- 入力タイプ：テキスト入力
- プレースホルダー：「記事のタイトルを入力してください」
- スタイル：
  - 幅：100%
  - パディング：px-4 py-3
  - ボーダー：グレー300・角丸xl
  - フォーカス：ring-2 ring-green-500
- バリデーション：必須入力

#### 概要入力
- ラベル：「概要」（必須マーク付き・赤500）
- 入力タイプ：テキストエリア
- プレースホルダー：「記事の概要を入力してください（2-3文程度）」
- 行数：3行
- スタイル：
  - 幅：100%
  - パディング：px-4 py-3
  - ボーダー：グレー300・角丸xl
  - フォーカス：ring-2 ring-green-500
  - リサイズ：不可（resize-none）
- バリデーション：必須入力

#### カテゴリ選択
- ラベル：「カテゴリー」（必須マーク付き・赤500）
- 入力タイプ：セレクトボックス
- オプション：
  - 「ChatGPT」（デフォルト選択）
  - 「n8n」
  - 「Python」
  - 「その他」
- スタイル：
  - 幅：100%
  - パディング：px-4 py-3
  - ボーダー：グレー300・角丸xl
  - フォーカス：ring-2 ring-green-500
- バリデーション：必須選択

#### タグ入力
- ラベル：「タグ」
- 入力タイプ：タグ入力フィールド + 追加ボタン
- プレースホルダー：「タグを入力してEnter」
- 追加ボタン：
  - テキスト：「追加」
  - 色：グリーン600背景・白文字
  - スタイル：角丸xl
  - ホバー：グリーン700
- 表示：追加されたタグを削除可能なチップ形式で表示
  - スタイル：グレー100背景・グレー700文字・角丸full
  - テキスト：`#タグ名`
  - 削除ボタン：×アイコン（グレー500、ホバーで赤600）
- 操作：
  - Enterキー押下で追加
  - 追加ボタンクリックで追加
  - ×ボタンクリックで削除

#### アイキャッチ画像URL入力
- ラベル：「アイキャッチ画像URL」
- 入力タイプ：URL入力 + 画像アイコンボタン
- プレースホルダー：「https://example.com/image.jpg」
- 画像アイコンボタン：
  - アイコン：画像アイコン（Image）
  - 色：グレー700
  - スタイル：ボーダー・角丸xl
  - ホバー：グレー50背景
- プレビュー表示：
  - URL入力時に自動表示
  - スタイル：角丸xl・高さ48（h-48）・オブジェクトカバー
  - マージン：上3（mt-3）

## ③ 記事の構成（目次）セクション

### 目的
- 記事の階層構造を定義
- セクション・サブセクション形式で構成

### 表示項目
- セクションタイトル：「記事の構成（目次）」（`text-2xl font-bold`）
- 背景：白背景・角丸2xl・シャドウlg
- 「大項目を追加」ボタン（右上）：
  - 色：アンバー600背景・白文字
  - スタイル：角丸xl
  - アイコン：Plusアイコン
  - ホバー：アンバー700

#### 大項目（セクション）
- スタイル：
  - 左ボーダー：4px・アンバー500
  - 背景：アンバー50
  - 角丸：右側のみxl（rounded-r-xl）
  - パディング：p-6
- ラベル：「大項目 {番号}」（アンバー900・太字）
- 入力フィールド：
  - プレースホルダー：「例: 企業が得られるメリット」
  - スタイル：
    - 幅：100%
    - パディング：px-4 py-3
    - ボーダー：アンバー300・角丸xl
    - フォーカス：ring-2 ring-amber-500
    - 背景：白
- 削除ボタン：
  - アイコン：×アイコン（X）
  - 色：赤600
  - スタイル：角丸lg
  - ホバー：赤100背景

#### 中項目（サブセクション）
- 表示位置：大項目内の左マージン（ml-6）
- スタイル：
  - 背景：白
  - ボーダー：アンバー200・角丸xl
  - パディング：p-4
- ラベル：「中項目 {大項目番号}-{中項目番号}」（アンバー800・太字）
- タイトル入力：
  - プレースホルダー：「例: DEIを「実態」として示せる」
  - スタイル：
    - 幅：100%
    - パディング：px-4 py-2
    - ボーダー：グレー300・角丸xl
    - フォーカス：ring-2 ring-amber-500
- 内容入力：
  - ラベル：「内容」（グレー700・太字）
  - プレースホルダー：「この中項目の詳細な内容を入力してください...」
  - 行数：4行
  - スタイル：
    - 幅：100%
    - パディング：px-4 py-3
    - ボーダー：グレー300・角丸xl
    - フォーカス：ring-2 ring-amber-500
    - リサイズ：不可（resize-none）
- 削除ボタン：
  - アイコン：×アイコン（X）
  - 色：赤600
  - スタイル：角丸lg
  - ホバー：赤100背景

#### 「中項目を追加」ボタン
- 位置：各大項目内の下部
- 色：アンバー600ボーダー・アンバー600テキスト
- スタイル：2pxボーダー・角丸xl
- アイコン：Plusアイコン
- ホバー：アンバー50背景

### 動作
- 初期状態：大項目1つが表示（サブセクションなし）
- 「大項目を追加」ボタンクリック → 新しい大項目を追加
- 「中項目を追加」ボタンクリック → その大項目内に新しいサブセクションを追加
- 削除ボタンクリック → 該当項目を削除
- 入力値はリアルタイムで保持

## ④ アクションボタンセクション（下部）

### 目的
- 記事の投稿・キャンセル操作を提供

### 表示項目
- ボタン配置：右寄せ・横並び（flex justify-end gap-4）
- 「キャンセル」ボタン：
  - テキスト：「キャンセル」
  - 色：グレー700テキスト
  - スタイル：2pxボーダー・グレー300・白背景・角丸xl・太字
  - ホバー：グレー50背景
  - リンク：`/articles`
- 「投稿する」ボタン：
  - テキスト：「投稿する」
  - 色：グリーン600～エメラルド600グラデーション
  - スタイル：白文字・シャドウlg・角丸xl・太字
  - アイコン：保存アイコン（Save）
  - ホバー：shadow-xl + translateY(-1px)

### 操作・遷移
- 「キャンセル」ボタンクリック → `/articles` へ遷移
- 「プレビュー」ボタン（ヘッダー）クリック → アラート「プレビュー機能は開発中です」
- 「投稿する」ボタンクリック：
  - バリデーション実行（タイトル・概要が必須）
  - 成功時：アラート「記事を投稿しました！」→ `/articles` へ遷移
  - 失敗時：アラート「タイトルと概要は必須項目です」

## JavaScript機能

### 状態管理
```javascript
let tags = [];
let sections = [{ id: 1, title: '', subsections: [] }];
let sectionCounter = 1;
let subsectionCounters = {};
```

### タグ管理機能
- `addTag()`: タグ入力値を配列に追加し、再レンダリング
- `removeTag(tag)`: 指定タグを配列から削除し、再レンダリング
- `renderTags()`: タグ配列をHTML要素として描画

### セクション管理機能
- `addSection()`: 新しいセクションを追加
- `removeSection(id)`: 指定IDのセクションを削除
- `updateSectionTitle(id, title)`: セクションタイトルを更新
- `addSubsection(sectionId)`: 指定セクション内にサブセクションを追加
- `removeSubsection(sectionId, subsectionId)`: サブセクションを削除
- `updateSubsection(sectionId, subsectionId, field, value)`: サブセクションの値を更新
- `renderSections()`: セクション配列をHTML要素として描画

### 画像プレビュー機能
- 画像URL入力時に自動でプレビュー表示
- URLが空の場合はプレビューを非表示

### フォーム送信機能
- `handleSubmit()`: 
  - タイトル・概要のバリデーション
  - 成功時：アラート表示 + `/articles`へ遷移
  - 失敗時：エラーメッセージ表示

### プレビュー機能
- `handlePreview()`: 「プレビュー機能は開発中です」アラート表示

### イベントリスナー
- タグ入力フィールドでEnterキー押下 → `addTag()`実行
- 画像URL入力フィールド入力 → リアルタイムプレビュー表示
- フォーム送信 → `handleSubmit()`実行（デフォルト動作を防止）

## バリデーション

### 必須項目
- タイトル（title）
- 概要（excerpt）

### バリデーションルール
- タイトル・概要が空の場合：エラーメッセージ表示
- その他の項目は任意

---

## レスポンシブデザイン

### デスクトップ（lg以上: 1024px~）
- フォーム幅：最大1280px（max-w-5xl）中央揃え
- パディング：px-4 sm:px-6 lg:px-8

### タブレット（sm: 640px~1023px）
- パディング調整：px-6

### モバイル（~639px）
- パディング：px-4
- ボタンは縦積み可能

  title: string,

  imageUrl: string,

      title: string,

## 共通仕様

### アニメーション
- ボタンホバー時：`transform: translateY(-1px)` + `shadow-xl`（投稿ボタン）
- トランジション時間：300ms（transition-all duration-300）

### 色指定
- 主要カラー：グリーン（`green-600`）、エメラルド（`emerald-600`）
- セクションカラー：アンバー（`amber-500`, `amber-600`）
- エラーカラー：レッド（`red-500`, `red-600`）
- 背景カラー：グレー50（`gray-50`）、白（`white`）

### フォント
- ページタイトル：`text-4xl font-bold`
- セクションタイトル：`text-2xl font-bold`
- ラベル：`text-sm font-semibold`
- 入力フィールド：`text-base`
