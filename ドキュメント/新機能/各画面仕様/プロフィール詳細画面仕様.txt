# プロフィール詳細画面 機能要件

## 概要
登録ユーザーの詳細プロフィールページ。スキル、経歴、提供サービス、ブログ記事などを表示する。ログイン不要で閲覧可能。

---

## ① ヘッダーセクション

### 目的
- プロフィール一覧への戻るリンクを提供

### 表示項目
- 戻るリンク：「プロフィール一覧に戻る」
  - アイコン：左向き矢印（ArrowLeft）
  - 色：グレー600
  - ホバー：グレー900
  - マージン下：mb-6

## JavaScript機能

### スムーズスクロール機能
```javascript
function scrollToSection(sectionId) {
  const element = document.getElementById(sectionId);
  if (element) {
    element.scrollIntoView({ behavior: 'smooth', block: 'start' });
  }
}
```

### スキル展開機能
```javascript
let showAllSkills = false;

function toggleSkills() {
  showAllSkills = !showAllSkills;
  const additionalSkills = document.getElementById('additionalSkills');
  const toggleText = document.getElementById('skillsToggleText');
  
  if (showAllSkills) {
    additionalSkills.style.display = 'flex';
    toggleText.textContent = '閉じる';
  } else {
    additionalSkills.style.display = 'none';
    toggleText.textContent = 'もっと見る（登録スキル数：8）';
  }
}
```

## アニメーション

### トランジション
- ボタン・リンク：transition-colors または transition-all
- 画像ホバー：transition-transform duration-500
- スムーズスクロール：behavior: 'smooth'

### ホバー効果
- リンク：テキスト色変更
- カード：シャドウlg
- 画像：scale-110
